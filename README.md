# A2SV Solved Questions 🚀

This repository contains my solutions to Data Structures and Algorithms (DSA) problems solved as part of the **A2SV (Africa to Silicon Valley)** training program.

## 📚 Purpose
- Track daily DSA progress
- Practice problem-solving and algorithmic thinking
- Prepare for technical interviews
- Submit solutions to the A2SV progress sheets

## 🛠️ Tech Stack
- Language: Python
- Focus: Data Structures, Algorithms, Time & Space Complexity

## 📂 Structure
Each problem is stored in a separate file named after the question, containing a clean and optimized solution.

## ⏱️ Progress Tracking
For each solved problem, the solution link and time taken are submitted to the respective A2SV group progress sheet.

---

✨ Consistency over speed. Growth over perfection.
